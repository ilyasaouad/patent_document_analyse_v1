Patent Claims

PATENT CLAIMS (BROAD VERSION – CLAIMS)

1. A method for operating a multimodal, adaptive virtual companion system, characterized in that the system:

1. a) receives the user’s speech and analyzes speech data in real time,

2. b) generates a response based on both social dialogue and expert knowledge,

3. c) automatically adapts conversation style, function, and role based on detected context,

4. d) is activated via voice command without physical contact,

5. e) identifies the user using biometric signals, and

6. f) monitors critical events and initiates assistance measures.

2. A system for carrying out the method of claim 1, characterized in that the system comprises at least: a speech interface module, an adaptive persona simulator, an expert module, a multimodal sensor module, a biometric identification module, a safety detection module, and a security architecture with local data processing.

3. The system according to claim 1, characterized in that it provides human-like social dialogue in friend mode.

4. The system according to claim 1, characterized in that it generates professional advice in expert mode.

5. The system according to claim 1, characterized in that it automatically switches roles between functions.

6. The system according to claim 1, characterized in that it provides interview simulation and presentation training.

7. The system according to claim 1, characterized in that it uses local wake-word processing.

8. The system according to claim 1, characterized in that it enables remote conversation via a microphone array.

9. The system according to claim 1, characterized in that it performs local facial and voice recognition.

10. The system according to claim 1, characterized in that it provides sleep-onset support through storytelling or audio.

11. The system according to claim 1, characterized in that it provides personal waking through a generated voice profile.

12. The system according to claim 1, characterized in that it generates child-friendly content.

13. The system according to claim 12, characterized in that it alerts guardians in the event of restlessness or crying.

14. The system according to claim 1, characterized in that it detects falls via multisensor analysis.

15. The system according to claim 1, characterized in that it automatically sends an emergency notification to emergency services or contacts.
